vcash_hash (v1.1)
===========================

Python module for Vcash's Proof-of-Work hashing (Whirlpoolx & Blake256 8rounds).


Install
-------

Python 2.7 is required as well as gcc.

    $ sudo python setup.py install


Test
-------

After installation, test hash.

    $ python test.py
